Hướng dẫn Break Link - lột trần bộ mặt những ông link "cứng đầu"

Nếu bạn là một con auditor thuần chủng, hoặc một accountant chính hiệu, thì chắc hẳn bạn đã đôi lần bát gặp một vài cái link không thể break.Vậy hãy tham khảo bài viết dưới đây, chỉ với vài click cùng với khả năng Copy-Paste thiên bẩm, bạn sẽ loại bỏ được những cái link cứng đầu kia trong nháy mắt. Let's gô.

Bước 1: Bấm tổ hợp Alt + F11 để mở cửa sổ Microsoft Visual Basic for Application (VBA)

Tại màn hình này, chúng ta cần thấy được panel Project Explorer để tìm đến file cần Break Link.

Nếu panel này chưa hiện ra, hãy ấn tổ hợp phím Ctrl + R, hoặc vào menu View > Project Explorer

Bước 2: Right-click 1 sheet bất kỳ của file cần break link, chọn Insert > Module

Copy đoạn code dưới đây, paste vào module vừa tạo và bấm F5

Sub unhidename()
For Each n In ThisWorkbook.Names
n.Visible = True
Next n
End Sub

Bây giờ, hãy quay lại file excel chứa link, vào Formulas > Name Manager, delete toàn bộ đống name vô tích sự kia đi.

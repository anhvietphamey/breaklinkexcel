Hướng dẫn Break Link - lột trần bộ mặt những ông link "cứng đầu"

Nếu bạn là một con auditor thuần chủng, hoặc một accountant chính hiệu, thì chắc hẳn bạn đã đôi lần bắt gặp một vài cái link không thể break.Vậy hãy tham khảo bài viết dưới đây, chỉ với vài click cùng với khả năng Copy-Paste thiên bẩm, bạn sẽ loại bỏ được những cái link cứng đầu kia trong nháy mắt. Let's gô.


Nguyên nhân của các link mà break xong vẫn còn nguyên là do file bạn đang dùng đã được kế thừa từ nhiều đời, hoặc bạn đã move sheet từ một file của tiền bối hoặc từ PBC,...mà trong đó có các name tham chiếu đến những file khác bên ngoài và các name đã bị hidden. Các name này không hiện ra khi bạn mở cửa sổ Name Manager. Vì vậy, để break link thì trước tiên cần make it visible.

Bước 1: Bấm tổ hợp Alt + F11 để mở cửa sổ Microsoft Visual Basic for Application (VBA)

Tại màn hình này, bạn cần thấy được panel Project Explorer để tìm đến file cần break link.

Nếu panel này chưa hiện ra, hãy ấn tổ hợp phím Ctrl + R, hoặc vào menu View > Project Explorer

Bước 2: Right-click 1 sheet bất kỳ của file cần break link, chọn Insert > Module

Copy đoạn code dưới đây, paste vào module vừa tạo và bấm F5

Sub UnhideNames()
For Each n In ThisWorkbook.Names
n.Visible = True
Next n
End Sub

Bây giờ, hãy quay lại file excel chứa link, vào Formulas > Name Manager. Loạt name vô nghĩa đã hiện lên, hãy xoá hết đi.

Sau đó check lại xem chú link kia còn đấy không? Hết rồi chứ sao nữa! (Còn nữa thì cũng chịu, không chữa được)

Cuối cùng, đừng quên xoá module bạn vừa tạo để có thể save được file dưới dạng .xlsx

Right-click vào tên module, chọn Remove


Chọn No khi được hỏi.

Còn nếu bạn vẫn muốn giữ lại đoạn code kia thì cần save file với extension .xlsm hoặc .xlsb

That's it.
